Modulação com JavaScript

Em JavaScript, a modulação permite que nosso código seja dividido em partes menores e reutilizáveis, melhorando a manutenção e a organização do código.

Exemplo de Exportação e Importação com ES Modules

Aqui, dividimos funções em diferentes módulos e as importamos em um arquivo principal:


// Arquivo math.js
export function somar(a, b) {
    return a + b;
}
export function subtrair(a, b) {
    return a - b;
}

// Arquivo main.js
import { somar, subtrair } from './math.js';
console.log(somar(10, 5));  // Saída: 15
console.log(subtrair(10, 5));  // Saída: 5

Organizando Código com Múltiplos Módulos

Dividir funcionalidades em módulos especializados ajuda a manter cada parte focada em uma única responsabilidade.


// Arquivo calculadora.js
export function multiplicar(a, b) {
    return a * b;
}
export function dividir(a, b) {
    return a / b;
}

// Arquivo app.js
import { somar, subtrair } from './math.js';
import { multiplicar, dividir } from './calculadora.js';

console.log(multiplicar(6, 7)); // Saída: 42
console.log(dividir(42, 7));    // Saída: 6
